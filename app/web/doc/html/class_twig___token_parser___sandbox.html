Detailed Description

Marks a section of a template as untrusted code that must be evaluated in the sandbox mode.

{% sandbox %}
    {% include 'user.html' %}
{% endsandbox %}
See Also
http://www.twig-project.org/doc/api.html#sandbox-extension for details
