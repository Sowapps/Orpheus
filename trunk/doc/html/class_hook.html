triggerHook (   $params = NULL)
Parameters
$paramsA callback.
Returns
The first param as result.
See also
trigger()

Triggers this hook calling all associated callbacks. $params array is passed to the callback as its arguments. The first parameter, $params[0], is considered as the result of the trigger. If $params is not an array, its value is assigned to the second value of a new $params array.
