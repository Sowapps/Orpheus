Parameters
$dateThe date to check.
$withTimeTrue to use datetime format, optional. Default value is false.
$timeThe output timestamp of the data, optional.
$countryThe country to use the date format, optional. Default and unique value is FR, not used.
Returns
True if $date si a valid date.

The date have to be well formatted and valid. The FR date format is DD/MM/YYYY and time format is HH:MM:SS Allow 01/01/1970, 01/01/1970 12:10:30, 01/01/1970 12:10 Fill missing informations with 0.
